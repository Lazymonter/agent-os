# 第一轮实现前必须具备的内容

## 项目侧必须具备

- 项目 `AGENTS.md`。
- Project Pack。
- 一期范围和非范围。
- 红线规则。
- 源资产登记。
- 第一轮 `goal.yaml`。
- Agent OS 锁定版本。

## Agent OS 侧必须具备

- `project-bootstrap` Skill。
- `iteration-start` Skill。
- `agent-engagement` Skill。
- `task-decomposition` Skill。
- `task-execution` Skill。
- `qa-verification` Skill。
- `iteration-close` Skill。
- 任务包模板。
- 评审标准。
- 关闭迭代策略。

## 不建议第一轮就做

- 大范围 UI 实现。
- 复杂发布流程。
- 多端并行实现。
- 未冻结 schema 的跨端联调。
