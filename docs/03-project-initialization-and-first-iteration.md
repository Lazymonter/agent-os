# 项目初始化与第一轮迭代

## 项目初始化目标

项目初始化不是写代码，而是建立项目能够被 Agent 执行的基本结构，包括：

- 项目规则 `AGENTS.md`。
- Project Pack。
- `.agent/agent-config.yaml`。
- `.agent/agent-os.lock`。
- 源资产登记。
- 初始迭代 `I-0000-project-initialization`。

## 第一轮迭代目标

第一轮迭代建议选择范围小、基础性强、可测试的功能，例如：

- 配置 schema 与校验。
- CLI validate。
- 原始数据写入。
- 核心评分函数。

## 第一轮迭代必须产出

- `goal.yaml`
- `iteration-brief.md`
- `agent-engagement-plan.yaml`
- `source-artifact-trace.md`
- `requirements.md`
- `architecture/task-decomposition.md`
- `task-packets/`
- `reviews/`
- `decision-pack.md`
- `learning/`

## 第一轮结束必须执行

- QA 验证。
- 必要的安全和契约审查。
- 人类决策。
- `iteration-close`。
