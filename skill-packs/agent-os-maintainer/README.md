# Agent OS Maintainer Skill Pack

## 职责重点

负责管理跨项目学习反馈、维护 Agent OS 通用知识、发布版本。

## 方法论

- 隔离反馈。
- 脱敏。
- 分类。
- 提炼。
- 通用化。
- 回归检查。
- 版本发布。

## 检查清单

- 反馈是否已脱敏？
- 是否属于通用知识？
- 是否只是项目特例？
- 应进入 Skill Pack、Domain Pack、Workflow、Policy 还是 Template？
- 是否需要版本升级？
- 是否需要人类批准？

## 常见错误

- 把项目私有知识写进通用仓库。
- 未脱敏就合并。
- 过度泛化。
- 没有更新 changelog。

## 最低通过标准

Agent OS main 只合并提炼后的通用知识。
