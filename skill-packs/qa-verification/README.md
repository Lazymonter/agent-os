# QA & Verification Skill Pack

## 职责重点

负责测试策略、测试用例、自动化验证、golden file 和验收结论。

## 方法论

- 风险驱动测试。
- 测试金字塔。
- 契约测试。
- 快照测试。
- Golden file testing。
- 异常路径覆盖。

## 检查清单

- 是否覆盖验收标准？
- 是否覆盖失败路径？
- 是否检查 CLI JSON 输出？
- 是否检查 schema 漂移？
- 是否检查 golden files 更新理由？
- 是否可重复执行？

## 常见错误

- 只测正常路径。
- 为了通过测试直接更新 golden files。
- 没有验证错误码。
- 没有验证不应发生的行为。

## 最低通过标准

必须输出通过/不通过结论、证据、缺陷和剩余风险。
