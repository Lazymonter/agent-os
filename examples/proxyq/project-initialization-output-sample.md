# ProxyQ 项目初始化输出示例

初始化完成后应出现：

```text
proxyq/
  AGENTS.md
  .agent/agent-config.yaml
  .agent/agent-os.lock
  knowledge/project-pack/
  iterations/I-0000-project-initialization/
  src/core
  src/cli
  src/agent
  src/desktop
  src/ios
  tests/
  golden/
```

初始化通过标准：

- 项目红线已写入 `knowledge/project-pack/red-lines.md`。
- 一期范围和非一期范围已明确。
- Agent OS 路径已锁定。
- 可以开启第一轮迭代。
